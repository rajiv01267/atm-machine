# atm-machine
A console-based simulation of an ATM machine that allows users to securely log in to their account, check balance, deposit, and withdraw money.

## Features
- Secure login with Account Number & PIN
- Balance Inquiry
- Deposit Amount
- Withdraw Amount
- Input Validation
- Robust Error Handling
